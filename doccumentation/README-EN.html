System Requirements

MySQL 5.6+ / MariaDB

PDO, mbstring, curl, fileinfo, openssl, json

Writable Directories

/assets/uploads/, /config/

Installation Steps

1. Upload Files

  1. Extract the full script archive to your computer
  2. Locate the file License_Key.txt containing your 22-digit license key
  3. Upload all files to your server using FTP or cPanel

2. Run the Installer

  1. Navigate to: http://yourdomain.com/install
  2. Follow the installer steps:
    • System Check (PHP version, extensions, writable paths)
    • Enter License Key (from License_Key.txt)
    • Database Setup: DB Host, Name, Username, Password
    • Site Configuration: Admin Username, Email, Password
  3. Complete the installation

3. Post-Installation

  1. Delete the /install/ folder manually for security
  2. (Optional) Import pre-made database from database/database.sql
  3. Delete the default user and create your own admin account

üîå API Endpoints

Located in /api/, these endpoints allow integration and automation:

POST /api/exchange.php - Submit an exchange request
GET /api/track.php - Track transaction via ID
GET /api/get_rates.php - Get current exchange rates
GET /api/get_post.php - Fetch blog posts
GET /api/get_server_time.php - Sync time between client/server
GET /api/generate_id.php - Generate unique transaction ID

üìú License System

The script includes a built-in licensing system to prevent unauthorized use:

  • License Key: Found in License_Key.txt (22 characters)
  • License Activation: Handled during installation
  • Protection Files:
    • /includes/license_check.php
    • /includes/license_protection.php
    • /config/license.php
  • Remote API: Validates the license key
  • Protection: If invalid, access to frontend and admin is restricted

‚ùì FAQ

Q: I get a "License invalid" error.
A: Make sure:
  • You entered the correct license key (22 digits)
  • Your server allows outbound connections (for license API)
  • You haven't already used the key on another domain (unless multi-use)
Q: How do I reset the admin password?
A: Use /admin/site-user/reset_password.php or update it directly in the database.
Q: Can I disable the blog or static pages?
A: Yes, simply remove or comment out routes like blog.php and hide them in the menu.
Q: How to back up my data?
A:
  1. Export your MySQL database via phpMyAdmin or command line
  2. Zip your full project files and download them
